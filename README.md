# OpenSCAD-PiezasDibujo
A library of 3D printable open source pieces to learn to draw perspectives and views. The pieces have been designed with OpenSCAD

# Files

Every piece have three files
- .scad openscad code 
- .stl 3D object 
- .png with an image
